En este proyecto de automatismo la finalidad y/o meta es generar un bot que me cancele tareas de PTS Acivo
por lo que estuve averiguando las herramientas que puedo utilizar son;

OpenPyXL: Permite leer el contenido de las celdas de Excel sin necesidad de tener el programa abierto.

PyAutoGUI: Simula movimientos del ratón, clics y pulsaciones de teclas para interactuar con cualquier software de escritorio que no tenga una API oficial.

Pyperclip: Útil para manejar el portapapeles y asegurar que caracteres especiales se copien correctamente.
